# YoQuieroSurvey
A Survey Taker for TB, for the people who want to leave a positive survey but are to lazy to actually do the survey

# Requirments
* Python 3.6.x
* Selenium 
* webdriver manager

# Installation
    pip install selenium
    pip install webdriver_manager
